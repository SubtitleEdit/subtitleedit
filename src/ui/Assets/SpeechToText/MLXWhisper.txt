MLX Whisper runs Apple's "mlx-whisper" Python library via python3.

Requirements
- An Apple Silicon Mac (M1 or newer).
- Install once:  pip3 install mlx-whisper

Why use it
MLX runs Whisper on the Apple GPU and Neural Engine via Apple's MLX framework,
so on M-series Macs it is typically much faster than the CPU-only
"Faster Whisper Mac" engine, at the same accuracy.

Models
tiny, base, small, medium, large-v2, large-v3, large-v3-turbo. These are
MLX-format Whisper weights hosted on Hugging Face (the mlx-community org) and are
downloaded automatically on first use.

Tip: large-v3-turbo is multilingual and the best speed/quality pick.

Advanced
The "command line parameters" box forwards any recognized mlx-whisper transcribe
option to the decoder. The option name is matched against the installed
mlx-whisper version, so unsupported options are reported rather than silently
dropped, and the value is coerced to the right type automatically.

Parameters (mlx-whisper transcribe options; type them into the box):

  --condition-on-previous-text N       [true   ] feed the previous window's text as
                                                  context; set False to stop repetition
                                                  loops and hallucinated credit lines
  --temperature N                      [0       ] sampling temperature; a single 0
                                                  disables the temperature fallback
  --compression-ratio-threshold N      [2.40   ] treat a window as repetitive above this
  --logprob-threshold N                [-1.00  ] retry a window below this confidence
  --no-speech-threshold N              [0.60   ] silence sensitivity
  --best-of N                          [5      ] candidates to sample on a retry
  --beam-size N                        [0       ] beam search width (0 = greedy)
  --patience N                         [       ] beam search patience
  --length-penalty N                   [       ] beam search length penalty
  --suppress-tokens LIST               [-1     ] token ids to suppress (-1 = default set)
  --suppress-blank N                   [true   ] suppress blank outputs at the start
  --max-initial-timestamp N            [1.00   ] cap on the first timestamp
  --clip-timestamps LIST               [0       ] comma separated start,end seconds to decode
  --prepend-punctuations LIST          ["'([{-  ] punctuation merged onto the next word
  --append-punctuations LIST           [.,!?:)  ] punctuation merged onto the previous word
  --hallucination-silence-threshold N  [       ] skip silence longer than this on a
                                                  hallucination (needs word timestamps)

The option names and value types are read from the installed mlx-whisper version,
so unsupported options are reported rather than dropped. Options Subtitle Edit
already controls (language, task, word timestamps, the initial prompt) are ignored
if typed here.

You can also type a full Hugging Face repo id as the model (anything containing
"/") to use a custom MLX Whisper model.

Progress
When the optional VAD (faster-whisper) is not installed, long files are decoded
in fixed windows so the console keeps showing progress instead of nothing for a
long time, and repetition loops cannot spread across the whole recording.
