Qwen3-ASR is a speech recognition system developed by Alibaba (Qwen team) that can transcribe audio and detect the spoken language.

The Qwen3-ASR family includes models such as Qwen3-ASR-0.6B, which support both automatic speech recognition (ASR) and language identification.

The Qwen3-Forced-Aligner-0.6B model can generate precise timestamps, making it useful for subtitle synchronization and alignment.

These models use GGUF quantization for efficient and fast inference, even on local machines.

Advanced settings are prepended before the required model/audio/output arguments.

Useful optional arguments:
-t 8          Set number of CPU threads

Project repository: https://github.com/woct0rdho/qwen3-asr.cpp
